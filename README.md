# gradle-kick-start steps to follow to create and build Gradle Project

cd project name

> gradle init

> ./gradlew help

> ./gradlew tasks

> ./gradlew compileJava

> ./gradlew processResources

> ./gradlew jar

> java -jar build/libs/gradle-strt.jar en

> ./gradlew test
